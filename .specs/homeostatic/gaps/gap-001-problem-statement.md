# Gap: Primary Problem Statement

## Question

What is the single-sentence problem this solves?

## Context

Need crisp articulation of the core pain point. This is the highest-level gap - almost everything else depends on understanding what problem we're solving.

## Resolution

**Problem:** Software systems are designed for creation and handoff, but serious work is 0.00001% creation and (infinity - 1)% long-term maintenance.

**Insight:** Nature's systems last forever because they are designed for continuous self-maintenance (homeostasis), not one-time creation. This pattern is mandatory for anything designed to last.

**Single sentence:** Homeostatic reconciliation enables systems to maintain themselves indefinitely by continuously detecting and closing gaps between desired state and actual state.
