# Gap: Target Users

## Question

Who uses this system? Developers building reconcilers? Agents running reconciliation? Both?

## Context

Need to understand if this is a library for developers to build reconcilers, or a runtime for agents to execute, or both. Affects how we describe the user story and goals.

## Resolution

**Users:** Tom + Bramwell + their agents.

**Scope:** Personal tool for niche elite mini-squads that work tightly together.

**Work modes supported:**

- Manual work (Bramwell)
- One-shot AI work
- Full agentic swarms
- Delegation to other people

Not a general-purpose library for the public (yet). Designed for a tight team that trusts each other and works closely.
